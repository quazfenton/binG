Setup Required

The Next.js web app requires Node.js to run. The sidecar server powers AI chat, agents, sandboxes, and LLM features. Local file operations work without it.

  1. Install Node.js 20+ if not already installed
  2. Open a terminal in the project web/ folder
  3. Run pnpm build to build the Next.js app
  4. Relaunch Quaz Desktop
